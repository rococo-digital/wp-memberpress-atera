Connect Memberpress to Atera knowledgebase and ticketing system

== Description ==

#### REQUIRES Memberpress #### https://memberpress.com/

When a user signs up a Customer and a contact is created in Atera

Allows importing tickets & knowledgebase from Atera

Can be combined with Live Search plugin: https://wordpress.org/plugins/search-live/ for instant search results (Custom post type = atera_knowledgebase).

Tickets can be created and viewed

Also adds ticketing 

== Installation ==

This section describes how to install the plugin and get it working.

1. install and activate memberpress: https://memberpress.com/
2. install and activate Atera Client Portal Extension
3. input Atera API in Atera CP menu: https://app.atera.com/Admin#/admin/api
4. import knowledgebase
5. install and activate Live Search: https://wordpress.org/plugins/search-live/
